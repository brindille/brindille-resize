# brindille-resize
Resize utils

## Install

With [npm](http://npmjs.org) do:

```bash
$ npm install brindille-resize --save
```

## Usage

```js
var resize = require('brindille-resize');
// add listener
resize.addListener(handler);
// remove listener
resize.removeListener(handler);
// get properties
var w = resize.width;
var w2 = resize.halfWidth;
var h = resize.height;
var h2 = resize.halfHeight;
```
